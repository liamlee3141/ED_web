# 必须配置的环境变量

1. 进入您的Supabase项目控制面板
2. 点击“设置（Project Setting）” → “Data API” → "Project URL"
3. 复制“项目URL”和“匿名公共密钥”
4. 匿名密钥 “Setting” →  "API Keys" → "API Keys"

# Your Supabase project URL
